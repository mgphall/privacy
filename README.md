# Privacy Policy Store

This repository serves as a centralized store for privacy policies.

## Structure

Privacy policies are organized in the `policies/` directory. Each policy should be stored as a markdown file with a descriptive name.

## Usage

To add a new privacy policy:
1. Create a new `.md` file in the `policies/` directory
2. Use a clear, descriptive filename (e.g., `app-name-privacy-policy.md`)
3. Follow the template structure provided in `policies/template.md`
